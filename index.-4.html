POLÍTICA DE PRIVACIDAD

El usuario siempre puede controlar sus conexiones ,eliminar los contenidos que dejen de interesarle y restringir con quien comparte sus conexiones.Queremos que comprendas que informacion recopilamos

Principio de transparencia lealtad y licitud: Se necesita el consentimiento expreso del usuario para poder realizar el tratamiento de datos personales. Principio de minimización: Solo se recabarán los datos imprescindibles para poder llevar a cabo los servicios solicitados por el usuario. Principio de limitación del plazo: Los datos personales se destruirán una vez se haya cumplido la finalidad para la que fueron recabados. Principio de confidencialidad e integridad: Se debe asegurar que se cumplen y toman todas las precauciones para que terceros no autorizados no puedan acceder a los datos.

Básicamente, cualquier página web, sea del tipo que sea, tiene la obligación de incorporar una página de política de privacidad en ella para cumplir con el RGPD y la LOPDGDD. Incluso un blog personal si tiene activados los comentarios, deberá contar con ella. Así mismo, este texto legal debe formar parte también de las políticas de privacidad de una empresa que tenga habilitada una página web.
